# 安全策略

## 支持的版本

| 版本 | 支持状态 |
|------|----------|
| main | ✅ 支持 |

## 报告漏洞

如果你发现安全漏洞，请通过以下方式报告：

1. **不要** 在公开 Issue 中披露漏洞细节
2. 请通过项目维护者的联系方式私下报告
3. 或者提交一个不包含敏感细节的 Issue，说明你发现了安全问题

我们会尽快回复并协调修复。

## 安全最佳实践

使用本项目时，请注意：

1. **本地部署**：本项目设计为全链路本地推理，请勿将服务暴露在公网
2. **敏感文件**：处理的法律文件包含敏感信息，请确保服务器安全
3. **模型文件**：模型权重应从官方渠道下载，避免使用不明来源的文件
4. **网络隔离**：建议在内网或 VPN 环境中使用

## 数据处理

- 本项目不会将任何数据上传到云端
- 所有推理均在本地完成
- 上传的文件存储在本地 `backend/uploads/` 目录
- 用户应自行管理和清理敏感文件
